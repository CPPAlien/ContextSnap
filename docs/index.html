ContextSnap

A tiny macOS menu-bar screenshot tool that pastes correctly into terminals, chat apps, and AI coding assistants — no thinking required.

Download for macOS
macOS 12.3+ · Apple Silicon & Intel · Free & open source

One hotkey, anywhere

Press ⇧⌘S to drag-select a region. Captures pile up in a translucent floating stack.

Smart paste

The same shot becomes a file path in your terminal, an image in Slack, and raw PNG bytes in Preview — receiver picks the format.

Built for AI workflows

Claude Code, ChatGPT, Cursor — drag a tile straight into the chat and it lands as the right kind of attachment every time.

Local-first

Every shot is archived under ~/Pictures/ContextSnap/. Nothing leaves your machine.

Install

  1. Download ContextSnap-x.y.z.dmg from the latest release.
  2. Open the DMG and drag ContextSnap.app into Applications.
  3. This build isn't notarized yet, so macOS will refuse to open it on first launch. Remove the quarantine flag once: 
    xattr -dr com.apple.quarantine /Applications/ContextSnap.app
  4. Launch ContextSnap. On the first capture, grant Screen Recording permission in System Settings → Privacy & Security.

Use it

  • ⇧⌘S → drag-select a region → it appears in the floating stack.
  • Drag a tile into any app to drop it.
  • Click a tile to copy it to the clipboard in every format at once.
  • Menu bar icon → Clear Stack wipes the overlay; Settings… lets you rebind the hotkey and change the save folder.
